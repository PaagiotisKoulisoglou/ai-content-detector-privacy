Privacy Policy for AI Content Detector Extension

Last Updated: January 2025

1. Information We Collect

The AI Content Detector extension collects and processes the following information:

  • Text Content: The extension extracts text content from web pages (emails, messages, video transcripts) that you actively choose to analyze by clicking the "Analyze Content" button.
  • Extension Settings: Your configured API endpoint URL is stored locally in your browser using Chrome's storage API.
  • No Personal Data: We do not collect personal information such as names, email addresses, or browsing history.

2. How We Use Your Information

The extracted text content is sent to the AI detection API endpoint that you configure in the extension settings. This API endpoint is under your control or the control of the service provider you choose.

  • Content Analysis: Text content is sent to your specified API endpoint for AI detection analysis.
  • Local Storage: Settings are stored locally on your device and are not transmitted to any third party.

3. Data Transmission

When you click "Analyze Content", the extracted text is sent via HTTPS POST request to the API endpoint you have configured. This transmission is:

  • Only sent when you explicitly click the "Analyze" button
  • Sent directly to your configured API endpoint
  • Not intercepted, logged, or stored by the extension itself

4. Third-Party Services

The extension allows you to configure your own AI detection API endpoint. The privacy practices of this third-party API service are governed by their own privacy policy. We recommend reviewing the privacy policy of your chosen API provider.

5. Data Storage

The extension stores the following data locally on your device:

  • Your configured API endpoint URL (in Chrome's sync storage)
  • No text content or analysis results are stored permanently
  • All processing is done in real-time

6. Permissions Used

The extension requests the following permissions:

  • activeTab: To access content on the current tab when you click the extension icon
  • scripting: To inject content scripts for text extraction
  • storage: To save your API endpoint configuration locally
  • host_permissions: To access web pages for content extraction and to send requests to your API endpoint

7. Your Rights

You have the following rights:

  • Control: You control when content is analyzed - it only happens when you click "Analyze Content"
  • Configuration: You choose which API endpoint to use
  • Removal: You can uninstall the extension at any time, which removes all locally stored settings
  • No Tracking: The extension does not track your browsing behavior or collect analytics

8. Children's Privacy

This extension is not intended for children under the age of 13. We do not knowingly collect information from children.
